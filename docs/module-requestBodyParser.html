    (require("requestBodyParser"))(contentType, body) → {string}

    Very simple, very lightweight, very light-on-features API that attempts to parse the body of a Ajax Request. You can pass it an object, in which case the parse will try to convert it to the correct type based on the contentType or a String, in which case the parser really does nothing other than passing back this string (it assumes you know what you are doing in that case).

    Example
    import parser from 'requestBodyParser'

const jsonData = parser('application/json', {foo: bar})
console.log(jsonData) // "{'foo': 'bar'}"

const plainData = parser('text/plain', {foo: bar})
console.log(plainData) // 'foo=bar'
